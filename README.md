# 🏋️‍♂️ Meu Treino - Projeto Fitness

Site pessoal desenvolvido para acompanhar minha rotina de treinos com foco em **hipertrofia muscular** e **redução de gordura corporal**.

---

## 📋 Sobre o projeto

Este projeto foi criado para facilitar o acesso ao meu plano de treino de forma prática, visual e responsiva, podendo ser utilizado tanto no computador quanto no celular durante os treinos na academia.

---

## 🎯 Objetivos

* Ganho de massa muscular (hipertrofia)
* Redução de gordura corporal
* Melhora do condicionamento físico
* Organização da rotina semanal de treinos

---

## 📅 Divisão de treino

| Dia     | Treino                             |
| ------- | ---------------------------------- |
| Segunda | Perna (Quadríceps - foco)          |
| Terça   | Superior (Peito + Costas + Braços) |
| Quarta  | Cardio + Abdômen                   |
| Quinta  | Superior (Ombro + Costas)          |
| Sexta   | Perna (Posterior + Glúteo)         |
| Sábado  | Cardio leve / atividade            |
| Domingo | Descanso                           |

---

## 💪 Estrutura dos treinos

* Exercícios compostos (base do treino)
* Exercícios isolados (refinamento muscular)
* Cardio estratégico para queima de gordura
* Divisão otimizada para recuperação muscular
